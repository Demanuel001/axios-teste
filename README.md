### Rodando o projeto
~~~bash
# Clone este repositório
$ git clone <git@github.com:Demanuel001/axios-teste.git>

# Dependências.
$ npm install react-icons
$ npm install axios

# Execute a aplicação em modo de desenvolvimento.
$ npm start

~~~
